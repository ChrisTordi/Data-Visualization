To learn more about software developers and their jobs, select between 2 and 5 attributes above and click
Submit.The order they appear in the attribute queue will be the order they are displayed in the visualization.

In this parallel sets chart you can see how proportions of software developers in each category change with each additional attribute. The width of each line is determined by the percentage of developers that fit into the next category that come from the total in the parent category. The width of each black line represents the percentage of all developers that fit into that subcategory. The colors correspond to the initial category from which a line came. Hovering over a line shows what percentage of the immediate parent category that line makes up.

The dataset used to make this visualization is the result of the Stack Overflow 2018 Developer Survey and the original dataset can be found here. It is important to note that the response rate for this survey from underrepresented groups in tech was lower than expected given their participation rates in the workforce.

This visualization used Jason Davies' parallel sets library which can be found here.
